Chords: Anonymous communication channels

This service allows you to configure a link that other users will use to send you a message using predefined list of contacts. The contacts would not be shared with requestor.

Important note

This is a pet project. By using this application, end user aggrees that he understands and accepts following risks:

  • Application can stop working at any time - don't use this app for emergency notifications
  • Someone can steal the data - application connects targets to chords and chords to user. Application remembers google account identifier, so chances are that it will be possible to map emails/phones with google account

How it works

The application provides different user expirience for users who want to send a message and for users who want to receive messages

I want to send the message

In order to send the message you need to know unique chord URL. Just open it in the browser, and application will give you a simple submission form. You don't need to be authenticated to send messages.

I want to receive messages

In order to receive messages you need to log in using Google Authentication and setup the chord in the "Chords" section. You will be able to setup the default notification text and one or many targets: emails or cell phone numbers. Application will give you unique chord URL associated with a particular cord. Application keeps target details in internal database and will not share it with senders.
